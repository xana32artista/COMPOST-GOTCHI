**Compost-gotchi** es una aplicación web interactiva y educativa pensada para enseñar el proceso del compostaje y la gestión sostenible de residuos orgánicos a través de mecánicas divertidas inspiradas en las mascotas virtuales (estilo *Tamagotchi*).

---

## ✨ Características Principales

- 🐛 **Mascota Virtual de Compostaje**: Cuida de tu propia compostera virtual equilibrando los niveles de humedad, aireación, temperatura y nutrientes (Nitrógeno y Carbono).
- ♻️ **Simulación Realista y Divertida**: Clasifica residuos orgánicos verdes (húmedos/nitrógeno) y marrones (secos/carbono).
- 🎮 **Gamificación y Progresión**: Gana puntos de experiencia (XP), sube de nivel y desbloquea insignias a medida que aprendes sobre desarrollo sustentable.
- 📱 **Diseño Moderno y Responsivo**: Interfaz fluida, atractiva y adaptada para computadoras, tabletas y dispositivos móviles.

---

## 🚀 Cómo ejecutar el proyecto localmente

El proyecto está construido 100% con tecnologías web estándar (**HTML5, Vanilla CSS3 y JavaScript puro**), por lo que no requiere instalación de librerías externas ni pasos complejos de compilación.

1. **Clona o descarga el repositorio**:
   ```bash
   git clone https://github.com/TU-USUARIO/TU-REPOSITORIO.git
   ```
2. **Abre la aplicación**:
   - Puedes abrir directamente el archivo `index.html` en cualquier navegador web moderno.
   - O iniciar un servidor web local (por ejemplo con Python o extensiones como *Live Server* en VS Code):
     ```bash
     python -m http.server 3000
     ```
     Y visitar `http://localhost:3000`.

## 🌐 Publicación en GitHub Pages

Para ver esta aplicación en vivo y compartirla con cualquier persona:

1. Sube este repositorio a tu cuenta de **GitHub**.
2. Ve a la pestaña **Settings** (Configuración) de tu repositorio en GitHub.
3. En el menú lateral izquierdo, selecciona **Pages** (Páginas).
4. En **Branch**, selecciona la rama `main` (o `master`) y la carpeta `/ (root)`.
5. Haz clic en **Save**. ¡En pocos segundos tu app estará disponible en una URL pública!
